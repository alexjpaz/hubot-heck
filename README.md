# hubot-heck

A hubot script that does the things

See [`src/heck.coffee`](src/heck.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-heck --save`

Then add **hubot-heck** to your `external-scripts.json`:

```json
[
  "hubot-heck"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
